# CafePosSystem

## 자바 콘솔 토이프로젝트<카페관리시스템>

## 사용 환경

#### Eclipse IDE 2021-03

#### JDK 11.02

## 주요 기능

#### 주문관리(주문기능)

#### 메뉴관리

#### 매출관리

#### 회원관리

---

## 진행

- 카페관리시스템을 구축하기 위한 더미 데이터 생성

- 메뉴관리, 카페에서 판매하는 메뉴의 CRUD기능

- 회원관리, 카페에 단골을 관리하기 위해 만든 메뉴

  - 회원 CRUD

  - 회원의 포인트, 쿠폰 수 관리

-관리자 로그인

-주문관리

  - 주문 내역(비회원)

  - 진행 중인 주문 취소

- 매출 관리

  - 연월일별 총 매출 출력

  - 음료 판매량 출력 진행중
