### 공통 기능

- 아이템 수에 따른 페이지 이동 기능

### 메뉴별 기능

- 메뉴관리, 카페에서 판매하는 메뉴의 CRUD기능

- 회원관리, 카페에 단골을 관리하기 위해 만든 메뉴
  - 회원 CRUD
  - 회원의 포인트, 쿠폰 수 관리

- 관리자 
  - 로그인
  - 로그아웃

- 주문관리
  - 주문 내역(비회원)
  - 진행 중인 주문 취소
  - 주문 내역(회원) : 포인트,쿠폰 적립, 쿠폰 사용

- 매출 관리(연월일별)
  - 매출액 출력
  - 음료 판매량 출력 
  - 쿠폰 사용에 따른 할인금액 출력
